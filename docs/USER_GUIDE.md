# Guide utilisateur ONYX

## Premiers pas

### 1. Configuration initiale

Au premier lancement :

1. Choisissez la longueur de votre code PIN (4 ou 6 chiffres).
2. Saisissez et confirmez votre code PIN.
3. (Optionnel) Activez la biométrie (empreinte ou Face ID) pour déverrouiller plus vite.

### 2. Créer un compte

1. Onglet **Comptes** → **+ Nouveau compte**.
2. Choisissez le type (Courant, Épargne, Cash, Investissement).
3. Nom et solde initial, couleur et icône.
4. Enregistrez.

### 3. Ajouter une transaction

**Rapide :** Bouton **+** → Dépense ou Revenu → montant → valider.

**Complète :** Transaction → Ajouter → remplir les champs (compte, catégorie, description, date, etc.).

### 4. Budgets et objectifs

- **Budgets :** Onglet Budgets → définir un plafond par catégorie (ex. Alimentation).
- **Objectifs d’épargne :** Onglet Objectifs → nom, montant cible, échéance.

## Sécurité

### Changer le code PIN

1. **Plus** → **Paramètres** → **Sécurité** (ou « Changer le code PIN »).
2. Entrez le code actuel.
3. Saisissez le nouveau code puis confirmez-le.

### Option « Effacer après 10 échecs »

Dans **Paramètres** → **Sécurité** :

- Si activée : après **10 codes PIN incorrects**, toutes les données de l’app sont **définitivement effacées**.
- À n’utiliser que si vous faites des sauvegardes régulières (export JSON).

### Sauvegarde (export)

1. **Plus** → **Données** (ou section dédiée).
2. **Exporter en JSON**.
3. Enregistrez le fichier dans un lieu sûr (cloud, ordinateur).

### Restauration (import)

1. **Plus** → **Données** → **Importer depuis JSON**.
2. Choisissez le fichier de sauvegarde.
3. Confirmez : **les données actuelles seront remplacées**.

## FAQ

**Mes données sont-elles dans le cloud ?**  
Non. ONYX fonctionne entièrement hors ligne ; tout reste sur l’appareil.

**Que faire si je perds mon téléphone ?**  
Sans export préalable, les données sont perdues. Pensez à exporter régulièrement.

**Plusieurs appareils ?**  
Oui, en important le même fichier JSON sur chaque appareil (export sur un, import sur l’autre).

**J’ai oublié mon code PIN.**  
Il n’est pas récupérable. Il faudra réinstaller l’app (les données seront perdues sans backup).

**L’app envoie-t-elle des données à l’extérieur ?**  
Non. Aucune donnée n’est envoyée à des serveurs.
